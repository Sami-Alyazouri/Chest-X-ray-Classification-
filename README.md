# Chest X-ray Classification (COVID-19, Pneumonia, Normal)

## 📌 Overview
This project presents a deep learning-based approach for classifying chest X-ray images into three categories: COVID-19, Pneumonia, and Normal. The goal is to assist in early detection and support medical diagnosis using AI techniques.

## 🧠 Model
- Convolutional Neural Networks (CNNs)
- Multi-class classification (3 classes)

## ⚙️ Features
- Image preprocessing and normalization  
- Data augmentation to improve model performance  
- Multi-class classification  
- Model evaluation using standard metrics  

## 📊 Evaluation Metrics
- Accuracy  
- Precision  
- Recall  
- Confusion Matrix  

## 🛠️ Technologies Used
- Python  
- TensorFlow / PyTorch  
- NumPy, Pandas  
- Matplotlib  

## 🚀 Results
The model demonstrates strong performance in distinguishing between COVID-19, pneumonia, and normal cases.

## 📌 Future Work
- Apply transfer learning (ResNet, VGG)  
- Improve model accuracy with larger datasets
